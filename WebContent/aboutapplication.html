• This is an AngularJS application which is used to find the largest palindrome number in the given range.
  • The User needs to enter the Lower bound value i.e. his lower number and upper bound value i.e. his higher number.
  • Once you enter the two values and click the find button the largest palindrome number in the given inputs will be appeared in the text Area.
  • Palindromic number is a number that remains the same even though the number is reversed. i.e. it is a symmetrical number it looks the same in number when it is written in reverse order.
  • For this project I have used AngularJs Framework and Html,Css,bootstrap for the UI part i.e. look and feel is developed using Html,css,bootstrap.
